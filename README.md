## Legend
| Symbol | Meaning        |
|--------|---------------|
| `@`    | Player        |
| `M`    | Monster       |
| `W`    | Weapon        |
| `P`    | Potion        |
| `#`   | Wall          |
| `.`    | Empty Space   |
| `E`    | Exit          |

## Game Rules
- Use WASD to move.
- Starting HP is 100, each potion restores 20 HP.
- Max HP is 150.
- Monsters have a random HP between 30 and 50 and do 10 damage per hit.
- Player's base damage is 10.
- Weapons increase damage by a modifier between 10 - 30.
- Highest damage modifier is always used.
- Game ends when HP reaches 0.

## Objective
- Your only goal is to reach the exit. 

## How to Build and Run
- Open the solution folder, build, and run AdventureGame.Cli
